# Reporting Guidelines

Use this file when the paper reports original research, evidence synthesis, a protocol, qualitative research, animal research, diagnostic accuracy, prediction modeling, intervention description, or clinical / health research.

Do not claim that a guideline has been followed unless the manuscript actually reports the required information. Use these guidelines as checklists for missing reporting elements and as routing cues for which reference file to read next.

## Guideline routing table

| Paper or study type | Guideline | Assistant should check |
|---|---|---|
| Randomized trial | CONSORT | trial design, participants, interventions, outcomes, sample size, randomization, allocation concealment, blinding, participant flow, harms, registration |
| Observational study | STROBE | design, setting, participants, variables, data sources, bias, study size, quantitative methods, missing data, limitations |
| Systematic review / meta-analysis | PRISMA 2020 | review question, eligibility criteria, information sources, search strategy, selection process, data collection, risk of bias, synthesis methods, certainty if used, flow diagram |
| Scoping review | PRISMA-ScR | rationale for scoping approach, eligibility criteria, information sources, selection process, charting process, synthesis of scope and gaps |
| Trial protocol | SPIRIT | trial rationale, objectives, design, eligibility, interventions, outcomes, sample size, allocation, blinding, data management, monitoring, ethics |
| Systematic review protocol | PRISMA-P | review question, eligibility criteria, information sources, search strategy draft, selection process, data extraction, risk-of-bias plan, synthesis plan, registration |
| Diagnostic accuracy study | STARD | participants, index test, reference standard, test methods, blinding, indeterminate results, accuracy estimates |
| Prediction model | TRIPOD / TRIPOD+AI | source data, participants, predictors, outcomes, sample size, missing data, model development, validation, performance, intended use |
| Case report | CARE | patient information, clinical findings, timeline, diagnostic assessment, intervention, follow-up, patient perspective if available, consent |
| Qualitative interview / focus group | COREQ | research team, reflexivity, study design, sampling, setting, data collection, analysis, findings, quotations |
| Qualitative research broadly | SRQR | qualitative approach, researcher characteristics, context, sampling, data collection, analysis, trustworthiness, synthesis |
| Animal study | ARRIVE | ethical statement, study design, sample size, inclusion/exclusion, randomization, blinding, outcome measures, statistical methods, animal details |
| Intervention description | TIDieR | intervention name, rationale, materials, procedures, provider, delivery mode, location, dose, tailoring, modifications, fidelity |

## How to use the routing

1. Identify the study type from the user's draft, title, abstract, methods, or journal instructions.
2. Read the corresponding guideline or checklist if the user supplied it; otherwise flag the likely guideline and list the manuscript elements to verify.
3. Do not turn guideline items into invented content. If an item is missing, ask for it or use a bracketed placeholder.
4. For mixed designs, combine the relevant guidelines. Example: an intervention trial with a detailed behavioral intervention may need CONSORT plus TIDieR.
5. For non-health fields, treat this table as a routing aid, not a universal requirement. Follow the target venue first.

## Common missing reporting items

- Trial registration, protocol registration, or preregistration identifier when applicable.
- Full search strategy and date of last search for evidence synthesis.
- Participant flow, screening flow, or study selection flow diagram.
- Sample size rationale.
- Blinding, allocation, randomization, or bias-control details.
- Risk-of-bias or quality-appraisal tool for reviews.
- Ethics approval, consent, or waiver details.
- Data, code, materials, or supplementary file availability.
- Intervention fidelity, tailoring, or modification details.
- Qualitative reflexivity, positionality, and representative quotations.
